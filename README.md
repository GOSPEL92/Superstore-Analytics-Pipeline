-- Superstore ETL, Python, SQL & Power BI Case Study

-- Project Overview
This project demonstrates an end-to-end analytics pipeline for the Superstore dataset, covering:
- Data extraction and cleaning with Python
- SQL transformations and queries
- Power BI dashboards for visualization
- Documentation and workflow diagrams in `/docs`

-- Tech Stack
- Python: Data cleaning, ETL scripts
- SQL: Querying and transformations
- Power BI: Interactive dashboards
- GitHub: Version control and collaboration

-- Folder Structure
- /DataPrep → Python scripts for cleaning and ETL  
- /SQL → SQL queries and schema definitions  
- /PowerBI → Power BI reports and dashboards  
- /docs → Diagrams, screenshots, and documentation  

-- Key Insights
- Sales trends across regions and categories  
- Profitability analysis by product segment  
- Customer behavior and order patterns
